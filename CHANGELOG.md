# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-XX

### Added
- Initial release of meAi Assistant plugin
- Customizable assistant UI with floating button
- Chat interface with typing animations and Lottie animations
- Support for custom logos, colors, fonts, and text
- Pre-built color schemes (meAi, purple, blue, green)
- Full color customization (17+ customizable colors)
- Image asset customization (logos, icons, animations)
- Authentication support (JWT tokens and SDK authentication with HMAC)
- API-suggested prompts with skeleton loading
- Custom suggestion prompts
- Keyboard-aware UI with automatic adjustments
- State management with MobX
- Multi-language support (en/ar)
- Typing indicators and message animations
- Skeleton loading for suggested prompts
- Customizable font family support
